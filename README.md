# TelemetryCollector_battery

## Description
Python tool to collect and organize telemetry data for ML model training.

## Features
- Collect battery and sensor data
- Save structured datasets for model training
- Configurable collection intervals

## Tech Stack
- Python 3.x
- pandas / CSV / JSON

## How to Run
1. Clone the repo  
2. Install dependencies: `pip install -r requirements.txt`  
3. Run: `python telemetry_collector.py`
